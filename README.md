# たそがれ日記（tasogare-diary）

1日の終わり（たそがれ時）に、言葉を選ぶだけで日記になる AI 駆動の日記コンパニオンアプリです。

- **技術スタック**: React Native / Expo、Firebase（Auth / Firestore / Functions）、Claude API
- **日記フロー**: きもち → できごと → ことば → たしかめる → 灯

## 📋 要件定義書（Notion）

本プロジェクトの**要件の正（source of truth）は Notion の要件定義書**です。機能・仕様の詳細は必ず下記を参照してください。要件の追加・変更は Notion を先に更新し、その後リポジトリのルールへ反映します。
